# Atividades da Disciplina: Técnicas de Programação

Este repositório é dedicado à documentação e ao armazenamento das atividades desenvolvidas na disciplina de **Técnicas de Programação**. O principal objetivo é apresentar de forma organizada os projetos, exercícios e trabalhos práticos elaborados ao longo do curso, demonstrando a aplicação dos conceitos teóricos em soluções de programação funcionais.

## 📚 Sobre a Disciplina

A disciplina de **Técnicas de Programação** aborda os fundamentos essenciais para a construção de software de qualidade, com foco em algoritmos, estruturas de dados e boas práticas de programação. A ementa da disciplina geralmente inclui:

* **Tipos Abstratos de Dados (TADs):** Implementação e uso de tipos de dados que encapsulam informações e operações.
* **Estruturas de Dados:**
    * Listas (sequenciais e encadeadas)
    * Pilhas
    * Filas
    * Árvores (binárias, de busca, etc.)
    * Grafos
* **Algoritmos de Ordenação e Busca:** Métodos como Bubble Sort, Quick Sort, Merge Sort, busca sequencial e busca binária.
* **Recursividade:** Resolução de problemas através de funções que chamam a si mesmas.
* **Alocação Dinâmica de Memória:** Gerenciamento de memória em tempo de execução.
* **Manipulação de Arquivos:** Leitura e escrita de dados em arquivos de texto e binários.
* **Modularização e Boas Práticas:** Organização do código em módulos, uso de cabeçalhos e documentação.

* **`Atividade-XX-NomeDaAtividade/`**: Cada pasta principal representa uma atividade, projeto ou lista de exercícios.
* **`src/`**: Contém o código-fonte da solução.
* **`doc/`**: Inclui a documentação, como o enunciado da atividade, diagramas ou relatórios.
* **`README.md`**: Um arquivo de descrição específico para cada atividade, explicando o problema, a solução implementada e como executá-la.

## 🚀 Como Executar os Projetos

Para compilar e executar as atividades deste repositório, siga as instruções abaixo.

### Pré-requisitos

* Certifique-se de ter o compilador ou interpretador da linguagem utilizada instalado em seu sistema.
* Um terminal ou prompt de comando.
* Opcional: A IDE recomendada para facilitar o desenvolvimento.

### Passos para Execução

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```

2.  **Navegue até a pasta da atividade desejada:**
    ```bash
    cd nome-do-repositorio/Atividade-XX-NomeDaAtividade/
    ```

3.  **Compile e execute o código:**
    As instruções detalhadas de compilação e execução para cada atividade estão no `README.md` específico da pasta. Por exemplo, para um projeto em C:
    ```bash
    gcc src/*.c -o executavel
    ./executavel
    ```
